# PPTX Design Principles

## Color Palette Selection

**Choosing colors creatively**:
- Think beyond defaults: Match colors to topic, not autopilot choices
- Consider: Topic, industry, mood, energy, audience, brand identity
- Build palette: 3-5 colors (dominant + supporting + accent)
- Ensure contrast: Text must be readable on backgrounds

## Visual Details Options

**Geometric Patterns**:
- Diagonal dividers, asymmetric columns (30/70, 40/60)
- Rotated text headers (90°/270°), circular/hexagonal frames
- Triangular accents, overlapping shapes for depth

**Border Treatments**:
- Thick borders (10-20pt) on one side only
- Double-line borders, corner brackets, L-shaped borders
- Underline accents (3-5pt thick)

**Typography**:
- Extreme size contrast (72pt vs 11pt)
- All-caps headers with wide letter spacing
- Monospace for data/stats, outlined text for emphasis

**Chart Styling**:
- Monochrome with single accent color
- Horizontal bars, dot plots, minimal gridlines
- Data labels directly on elements, oversized key numbers

**Layout Patterns**:
- Full-bleed images with overlays, sidebar columns (20-30%)
- Modular grids (3x3, 4x4), Z/F-pattern flow
- Magazine-style multi-column layouts

**Backgrounds**:
- Solid color blocks (40-60%), gradients (vertical/diagonal only)
- Split backgrounds, edge-to-edge bands, negative space
